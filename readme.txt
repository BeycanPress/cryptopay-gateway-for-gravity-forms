### Cryptocurrency Payment Gateway for Gravity Forms by CryptoPay

If you're using Gravity Forms on your WordPress site to collect data from users or sell products, and you'd like to receive payments in both FIAT currency and cryptocurrencies, you've found the perfect plugin. CryptoPay is a plugin developed specifically for WordPress, supporting over 15 WordPress plugins in addition to Gravity Forms.

Thus, CryptoPay is the ideal plugin for accepting crypto payments on forms created with Gravity Forms.

== Installation ==

1. Upload the entire `/cryptopay-gateway-for-gravity-forms` directory to the `/wp-content/plugins/` directory.
2. Activate CryptoPay Gateway for Gravity Forms through the 'Plugins' menu in WordPress.
3. Enjoy.
